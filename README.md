# TRON & Ethereum Vanity Address Generator

High-performance, secure, client-side vanity address generator for TRON and Ethereum.

## Security
- **Local Generation**: No data is ever sent to a server.
- **Offline Use**: For maximum safety, download this repo and run `index.html` while disconnected from the internet.

## Libraries
- [TronWeb](https://github.com/tronweb3/tronweb) (TRON)
- [Ethers.js](https://github.com/ethers-io/ethers.js) (Ethereum)
- [Libauth](https://github.com/bitauth/libauth) (High-speed WASM crypto)
